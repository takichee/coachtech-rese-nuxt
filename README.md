# Rese
飲食店予約アプリケーションです。ユーザー登録・飲食店の検索・お気に入り・予約をすることができます。
<img width="1334" alt="Screen Shot 2022-02-04 at 8 53 47" src="https://user-images.githubusercontent.com/65226658/152448782-c880da27-e84d-4745-ae51-b3e717b3f5d9.png">

## 作成した目的
Nuxt.jsとFirebase Authenticationを使用したユーザー管理の練習で作成しました。

## アプリケーションURL
https://wizardly-cori-f597b4.netlify.app/

メール認証はつけていないので、適当なメールアドレスとパスワードで登録可能です。

## バックエンドリポジトリ
https://github.com/takichee/coachtech-rese-laravel/tree/master

## 機能一覧
- ユーザー管理 (登録・ログイン・ログアウト)
- 店舗の検索 (エリア・ジャンル・店舗名)
- 店舗のお気に入り
- 新規予約・キャンセル
- マイページ機能 (予約履歴の表示・お気に入り店舗の一覧表示)
